We explore **AI Agents that go beyond traditional tool attributes**, creating new forms of intelligent hardware for real-world applications.

### Key Topics

- **Embodied AI agents** — designing agents that operate in physical environments with awareness and adaptability
- **Skill training systems** — AI-driven assistants for hands-on operational learning (e.g., emergency medical services)
- **Healthcare and rehabilitation** — personalized communication and care support for elderly and caregivers
- **Accessible interaction** — augmentative and alternative communication (AAC) tools, voice-based interfaces for users with visual impairments

### Representative Work

- *Hand by hand* — LLM driving EMS assistant for operational skill learning (IJCAI, 2025)
- *Pika* — designing a social-support agent to improve drivers' experience in gig work (IJHCS, 2025)
- *CareEmo* — supporting caregivers with personalized communication approaches (IEEE SMC, 2025)
- *SocializeChat* — a GPT-based AAC tool for social communication through eye gazing (UbiComp, 2023)
